## 검증 방법

```bash
uv run python scripts/check_stations.py
```

위 명령어 실행 결과를 보고 정식 stationName과 좌표(dmX=위도, dmY=경도)를
이 표에 기록한다. 매칭 실패한 측정소가 있으면 충북 전체 측정소 목록에서
실제 명칭을 찾아 `src/config.py`의 `TARGET_STATIONS`를 갱신한다.

## 측정소 변경 이력

- (초기 생성) 사용자 정의 4개 측정소 등록
