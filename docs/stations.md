## 검증 방법

```bash
uv run python scripts/check_stations.py
```

매칭 실패한 측정소가 생기면 충북 전체 측정소 목록에서 실제 명칭을 찾아
`src/config.py`의 `TARGET_STATIONS`를 갱신한다.
