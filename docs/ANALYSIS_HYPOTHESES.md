## 측정소별 위치 가설

- **오창읍 (북·개방지, 반도체)**: 반도체·이차전지는 클린룸 공정이라 입자 직접배출이 적다.
  그럼에도 PM10 피크가 높다면 산단 자체보다 **장거리 수송(외부 유입)의 1차 관문 + 개방 지형의 빠른 분산**이 원인.
- **오송읍 (서, 바이오·제약)**: 봄철 북서풍의 **풍상측**. 외부 기단이 가장 먼저 닿는다. 제약·바이오도 저배출 업종.
- **용암동 (동, 거주지)**: 산단 직접 배출이 없는 **대조군**. 새벽 상승이 가장 늦고 약해야 한다.
- **복대동/봉명동 (중앙, 산단육거리)**: 산업 + 교통 교차로 복합. 평상시 농도가 가장 높을 것으로 예상.

## 검증 대상 가설 3종

### H1 — 풍향(외부 유입) 가설
새벽 공통 상승이 기상 요인이라면, 봄 북서풍 기준 **서·북(오송·오창)이 동(용암)보다 먼저·높게** 반응해야 한다.
- 검정: 측정소 간 시차(lag) 상관 분석 + 기상청 풍향 API 결합 (Phase 4 `WEATHER_API`)

### H2 — 산단 근접도 가설
자체 배출이 지배적이라면 평균 농도가 **복대/봉명 > 오창/오송 > 용암** 순으로 갈려야 한다.
- 검정: 측정소(또는 그룹) 간 **ANOVA** + 사후검정 (Phase 2 `HYPOTHESIS_TEST`)

### H3 — 업종 저배출 가설 (H1·H2와 경쟁)
반도체·바이오·제약은 입자 직접배출이 적어, **산단 영향군과 베이스라인(용암)의 차이가 통계적으로 유의하지 않을 수 있다.**
- 차이 없음(H3 채택) → 새벽 피크는 산단이 아니라 **대기 경계층 일변화 + 교통**이 지배적이라는 강력한 증거
- 검정: **t-test** (산단군 평균 vs 용암동), 지표별(PM10/PM2.5 등) 반복

## 1일치 관측에서 본 단서 (결론 아님)

- 오창읍: 피크 최고(~60), 오후 급락(30) → 진폭 최대 (외부 유입 + 빠른 분산 가설과 부합)
- 오송읍: 새벽 동반 상승, 오후 완만 하락(43) → 유입 후 정체
- 용암동: 03→06시 상승 지연, 피크 최저(47) → 대조군 가설과 부합
- 복대/봉명: 데이터 부족(복대 통신장애) → 봉명동 누적 후 H2 검증
